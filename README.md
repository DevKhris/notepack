## About Notepack

NotePack is a Note Taking Management System developed with Laravel

## Getting Started

1. clone this repository

```
git clone https://github.com/DevKhris/notepack.git
```

2. Rename .env.example to .env

3. Change Database Credentials in .env

4. Run the migrations

```
php artisan migrate
```

5. Enjoy
